# Dr1 Gene Hospital Website

A premium, modern, multi-specialty clinical website for Dr1 Gene Hospital, featuring advanced diagnostic packages, digital test evaluations, medical scheduling slot information, and verified patient guides.

---

## 🚀 Key Features

- **Modern Single-Page UI/UX**: Centered on pristine medical design with Inter typography and custom accent colors.
- **Diagnostics Matrices**: Up-to-date packages and price listings for local Whitefield and Bangalore residents.
- **Modular Components**:
  - `Navbar` & `Footer` for fluid layout transitions.
  - `AboutView` featuring the qualifications and timings of medical consultants.
  - `ContactView` detailing physical directions, appointment bookings, and support structures.

---

## 🛠️ Tech Stack & Architecture

- **Framework**: React 18+ with TypeScript
- **Bundler**: Vite
- **Styling**: Tailwind CSS (loaded seamlessly via native imports)
- **Animations**: `motion` (by `motion/react`)
- **Icons**: `lucide-react`

---

## 📦 Getting Started

To run this project locally, ensure you have Node.js installed, then follow these steps:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Lint code quality**:
   ```bash
   npm run lint
   ```
